== Description ==

WordPress Easy Index will create an index for your WordPress Blog posts

== Installation ==

1. Upload the folder wp-easy-index to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Navigate to Manage > Option > Easy Index to configure plugin output.

== Upgrade ==

1. Delete the old plugin folder `wp-create-index`.
1. Upload the folder wp-easy-maps to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Navigate to Manage > Option > Easy Index to configure plugin output.
